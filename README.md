**[기본 개념]**

React : React JS 는 어플리케이션이 interactive 하도록 만들어 주는 library

React-dom : 모든 React element 들을 HTML Body에 둘 수 있도록 해주는 library 또는 package

JSX : JavaScript를 확장한 문법

Babel : JSX로 적은 코드를 브라우저가 이해할 수 있는 형태로 바꿔준다.
